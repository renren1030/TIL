#gitコマンドまとめ
```
git init
git にどのディレクトリを使うのかを宣言する
```
```
git add　ファイル名　or git add .
作業ディレクトリからステージングエリア（インデックス）にあげる
```
```
git commit
コミットする
ステージングエリアからリポジトリ（ローカル、リモート）にあげる
```
```
git commit -m "コミット名"
コミットのメッセージを一行に収める
```
```
git log 
コミット履歴の確認
```
```
git log --oneline
コミット履歴をコンパクトに一行で見れる
```
```
git log -p
どこが変更された場所か場所がわかる
```
```
git log --stat
どのファイルが何箇所変更されたかがわかる
```
```
git status
現在の状況を確認できる
```
```
git diff
どこを編集したのかがわかる
```
